Important Concepts

Lesson 1: The Basics of the Web and HTML

Web Structure

A web page is a text document written in a language called HTML (HyperText Markup Language).

HTML is made up of text content, markup (tags), references (images/videos) and links.

How Coding Works

Coding happens when programmers write text in a language that a computer can understand. The computer can the follow the instructions the programmer wrote. For example, we use tags to bold certain words in text like: I am learning to code!

Computers are Stupid

Programmers need to write exactly the way a computer understand (also known as writing with correct syntax)

For example, if you forget to close a formating tag, the computer will not be able to figure out what you had intended to format. This stupidity can be very frustrating, but it also gives programmers incredible power: if you know how to talk to a computer then you can it to do anything you want.

Programmers Can't Remember Everything

There are too many details to keep everything in your head. And that is okay. If you forget how to make text italic in HTML, you can always just look it up.

Basic HTML Vocabulary

Tag: An HTML tag is always contained within angled brackets. Most tags have an opening tag and a closing tab. Some tags (called void tags) do not require a closing tags.

Inline vs Block Tags

Inline Tags: Written in the line of the coded. Examples include formatting tags, line break, links, images and span tags.

Block Tags: Create and invisible box around text and can be used of styling purposes. Examples inlude paragraph tags and div tags.

Lesson 2: Creating a Structured Document with HTML

Developer Tools

HTML: The language that provides structure via elements that have syntax and rules

CSS: The language that provides style to elements that have been defined within the HTML text.

Java: The language that provides interactive components.

Elements

Elements are used to break-up websites as well as to provide reference points to add CSS styling.

Boxes

The structure of websites are based on boxes inside of boxes with different formatting and styling.

Websites can be made of any number of boxes. When creating a new site it is best to first figure out the structure of the site via how the boxes will be laid out.

Document Object Model

DOM or Document Object Model is the standard convention for interacting with elements in HTML. DOM provides the tree-like structure that is used in the coding.

Lesson 3: Adding CSS Style to HTML Structure

CSS

CSS: Also known as Cascading Style Sheets provide the formatting for HTML pages. Can use multiple CSS sheets or code the styling directly into the HTML code (either in the head section via a stlye element or inline)

Code, Test, Refine

1) Look for natural boxes.

2) Look for repetitive styles and semantic elements.

3) Write the HTML.

4) Apply Style with CSS (bigges to smallest).

5) Fix things by repeating steps 4 and 5 above.
